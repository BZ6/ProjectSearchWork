# Проект по поиску работы для джунов

## Запускаем в Docker

```bash
docker-compose up --build
```

## Инструменты

- Для CRUD используется: https://fastapi-crudrouter.awtkns.com/
- Для работы с ролями (авторизация, регистрация) будет юзаться: OAuth2
